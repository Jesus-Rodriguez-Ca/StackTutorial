How to create a web app with MEAN stack.

Step 1

Set up Ubuntu on Google Cloud

  1. * Visit the Google Cloud console page. *
  2. Create an account
  3. On the bottom right look for the link called   Console.
  4. On the top nav bar, select   Project.
  5. In the nav bar on the left find Compute Engine.
  6. Once you there select  VM Instance.
  7. Then   Create.
  8. Name the Instance
  9. Plugin your “Region” and “Zone” (Oregon)
  10. Change the “Machine type” to the smallest one “f1-micro”
  11. For “Bootdisk” Use “Ubuntu 18.04 LTS”
  12. Down at “Firewall” select “Allow HTTP Traffic” and “Allow HTTPS Traffic”
  13. Then hit “Create” and Wait
  14. Once its done loading check to see if ur instance is running by looking at the green dot with a white checkmark.
  15. Click “SSH” to the right of the instance that you want to use, keep in mind it will take some time to load.
  16. Continue to Node.js.

Step 2

Software

  1. Once the Command Prompt opens you’ll need to install Node.js, in the command line Paste these commands:
    2.  -sudo apt-get update

     - sudo apt-get install -y curl apt-transport-https ca-certificates && curl --fail -ssL -o
     - setup-nodejs https://deb.nodesource.com/setup_10.x &&
     - sudo bash setup-nodejs &&
     - sudo apt-get install -y nodejs build-essential
  2. To install Angular.js paste this into the command line.
    3.  - sudo -i npm install -g @angular/cli
  3. Type “y” or “n” and hit enter when asked to share data with Google (makes no difference for this tutorial)
  4. To install Express.js create a directory called “mean-stack”:
    5.  - mkdir mean-stack
  5. Then change the directory to helloworld.
    6.  - cd mean-stack
  6. After that, initialize your node files and your package.json.
    7.  - npm init
  7. On first prompt (package name): press Enter, on second prompt (version): press Enter, on third prompt (description): press, Enter.
  8. When asked for the “entry point” enter: app.js
  9. When asked for test command: press enter.
  10. When asked for git repository: press enter.
  11. Keywords: press enter
  12. Author: press enter
  13. License (ISC): press enter
  14. Is this OK? (yes): press enter
  15. Run express.js in your project and directory
    16.  - npm install express --save
  16. Finally, open the nano text editor:
    17.  - nano app.js
  17. Once you’re in nano copy and paste this below
    18. var express = require (’express’);
    var app = express();
    app.get(’/’, function(req, res) {
    res.send(’Hello world!’);
    });
    app.listen(3000, function () {
    console.log(’Example app listening on port 3000!’);
    });
  18. To exit the nano CTRL+X, then press y and enter
  19. Install MongoDB packages
    20.  - sudo apt install -y mongodb
  20. Run MongoDB
    21.  - sudo systemctl start mongodb
  21. To begin using mongo shell
    22.  - mongo
  22. To create and switch to a new collection named "mean-stack-db":
  23. Install MongoDB packages
    24.  - db.inventory.insertMany();

  25. Finally you can search stuff inside your database by creating queries. For instance:
    26.  - db.inventory.find( { status: "D"}

Step 3

Firewall Rule

  1. From command line:
    2.  - sudo iptables -I INPUT -p tcp --dport 3000 -j ACCEPT
  2. or
  3. Back in Google Cloud Platform, in the search bar search for: Firewall
  4. From the results select: Firewall rules
  5. Click on "CREATE FIREWALL RULE"
  6. Name: firewall-port-3000
  7. Description: Open port 3000
  8. Targets: All instance in the network
  9. Source IP ranges: 0.0.0.0/0
  10. Protocols and ports: Click on checkbox: "tcp" and in entry box type: "3000"
  11. Click CREATE button

Step 4

Running the server

  1. Back in the GNU Bash Shell run the server with the following command: (if still in the mongo db type exit
    2.  - node app.js
  2. If you see this you’re good
  3. Back in the Google Cloud Platform go to:
  4. Compute Engine
  5. Click on VM instances:
  6. Type the External IP address into the URL bar, removing the "s" from "https" and adding ":3000" to the end
  7. Example: http://34.82.213.84:3000
